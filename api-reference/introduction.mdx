## Authentication

All API endpoints are authenticated using Bearer tokens.<br />
The \<token\> in the Bearer token is the api_key you can get from: https://platform.ground.dev/dashboard/api-keys.

```json
{
  "Authorization": "Bearer <token>"
}
```
